4.3 多重判断

思考：中国合法工作年龄为18-60岁，即如果年龄小于18的情况为童工，不合法；

如果年龄在18-60岁之间为合法年龄；大于60岁为法定退休年龄。

age = int(input('请输入您的年龄：'))
if age<18:
    print(f'您的年龄是{age},童工一枚')
elif age>=18 and age<=60:
    print(f'您的年龄是{age},合法工龄')
elif age>60:
    print(f'您的年龄是{age},可以退休')

5 循环语句

5.1 while 结构
  • 例题1：计算1-100累加和

分析：1-100的累加和，即1 + 2 + 3 + 4 +….，

即前两个数字的相加结果+下一个数字(前一个数字+1)。

i=1
result=0
while i <= 100:
    result += i
    i += 1
print(result)

#返回 5050
  • 例题2：计算1-100偶数累加和

分析：1-100的偶数和，即2 + 4 + 6 + 8….，得到偶数的方法如下：

  • 偶数即是和2取余结果为0的数字，

    可以加入条件语句判断是否为偶数，为偶数则累加

  • 初始值为0 / 2 ,计数器每次累加2

#方法一：条件判断和2取余数为0则累加计算
i = 1
result = 0
while i <= 100:
    if i % 2 == 0:
        result += i
    i+=1
print(result)

#返回  2550
#方法二：计数器控制增量为2
i = 0
result = 0
while i <= 100:
    result += i
    i+=2
print(result)

#返回  2550
5.2 break/continue

举例：一共吃5个苹果，吃完第一个，吃第二个…，这里

“吃苹果”的动作是不是重复执行？

1、break

如果吃的过程中，吃完第三个吃饱了，则不需要

再吃第4个和第五个苹果，即是吃苹果的动作停止，

这里就是break控制循环流程，即终止此循环。

i=1
while i<= 5:
    if i == 4:
        print(f'吃饱了不吃了')
        break
    print(f'吃了第{i}个苹果')
    i+=1

# 返回
吃了第1个苹果
吃了第2个苹果
吃了第3个苹果
吃饱了不吃了

2、continue 结束本轮循环

如果吃的过程中，吃到第三个吃出一个大虫子…,

是不是这个苹果就不吃了，开始吃第四个苹
果，这里就是continue控制循环流程，

即退出当前一次循环继而执行下一次循环代码。

i = 1
while i <= 5:
    if i == 3:
        print(f'大虫子，第{i}个不吃了')
        # 这句一定要加上，否则陷入死循环，因为如果不加的话
        # 此时i=3,执行continue后，会跳到第二行，i=3，还是
        # 会进入到if判断语句中，加上后，i=4,就可以跳过if语句
        i += 1  
        continue
    print(f'吃了第{i}个苹果')
    i += 1

# 返回
吃了第1个苹果
吃了第2个苹果
大虫子，第3个不吃了
吃了第4个苹果
吃了第5个苹果

​ 3、重写

​ 1）定义：在子类中可以定义与父类相同名称的方法，此时子类的方法对父类构成了重写。

​ 2）如果子类重写了父类的方法，使用子类对象调用被重写的方法时，执行子类中重写后的方法。

例子：

class Animal:
    def eat(self):
        print("动物吃东西")

class Cat(Animal):
    def eat(self):
        print("猫吃鱼")

c=Cat()
c.eat()                   #返回值：猫吃鱼

​ 3）重写关系中调用父类的被覆盖的方法

  • super().方法名()不用写self

例子：

class Animal:
    def eat(self):
        print("动物吃东西")

class Cat(Animal):
    def eat(self):
        print("猫吃鱼")
        super().eat()

c=Cat()
c.eat()    #返回结果：猫吃鱼   动物吃东西
  • 父类名.方法名 Animal.eat()

4、多继承

​ class Child(Father,Mother)

​ 如果方法名重复，优先谁在前，调用谁

​ 父类名.方法名（self）

例子：

class Father:
    def sing(self):
        print("爸爸擅长唱歌")
    def dance(self):
        print("不喜欢跳舞")

class Mother:
    def sing(self):
        print("不擅长唱歌")
    def dance(self):
        print("妈妈喜欢跳舞")

class Child(Father,Mother):
    def sing(self):
        Mother.sing(self)

c = Child()
c.sing()           #返回结果：不擅长唱歌
8.6 多态

​ 定义：一个对象具有多种形态，在不同的使用环境中以不同的形态展示其功能，多态发生有不同关系的基础上。

​ 发生在继承的基础上。

例子：man就是一个多态

class Teacher:
    def teach(self):
        print("教授知识")

class Driver:
    def drive(self):
        print("开车")

class Man(Teacher,Driver):
    def teach(self):
        print("教授python知识")
    def drive(self):
        print("开小轿车")

class Demo:
    def play(self,driver):

           driver.drive()
man = Man()

d = Demo()
d.play(man)

一个类，形参，通常传的是一个变量，这里传的是一个对象，并且这个对象还有继承关系。
